
# Hospital CuidaMais

A Cuida+ é uma plataforma de telemedicina que une consultas online com especialistas e uma triagem inovadora por Inteligência Artificial (CuidaIA+) para agilizar o atendimento. O sistema democratiza o acesso à saúde integrando agendamentos de diversas especialidades, suporte a convênios e gestão de histórico médico, oferecendo cuidado completo na palma da mão.

Principais Funcionalidades:

1 - Telemedicina por Videoconferência: A plataforma permite a realização de consultas médicas diretamente por vídeo, conectando pacientes a profissionais de saúde de forma remota e segura.

2 - Triagem com Inteligência Artificial (CuidaIA+): Um dos diferenciais é o sistema CuidaIA+, uma IA "humanizada" que realiza uma triagem médica inicial. O paciente pode interagir via texto ou áudio (com opção de anexos) para descrever sintomas antes do atendimento humano.

 - A triagem analisa os sintomas e direciona o atendimento com maior precisão.

3 - Encaminhamento e Integração Física: Além do digital, a aplicação conecta o paciente ao mundo físico. Se necessário, o sistema encaminha o paciente para unidades de atendimento médico próximas e gera relatórios médicos automáticos após a consulta.

4 - Gestão de Saúde: O sistema centraliza o histórico do paciente e envia receitas médicas e orientações diretamente por e-mail, garantindo que todas as informações fiquem organizadas.

## Aprendizados

Construir o Cuida+ nos ensinou a conectar as pontas. Aprendemos como estruturar um front-end amigável (HTML/CSS/JS) e prepará-lo para consumir uma API robusta em Java (Spring Boot). Entendiemos a importância de planejar os endpoints para que o cadastro de pacientes e o agendamento conversassem perfeitamente com o banco de dados MySQL.

Ao implementar a página de login, aplicamos a integração de APIs de terceiros. Trabalhar com a biblioteca do Google Sign-In (vista no login.html) e manipulação de tokens (JWT decode) nos deu uma visão prática sobre como gerenciar sessões de usuário de forma segura, indo além do login básico.

Como é um site de hospital, o design precisava ser inclusivo. Aprendemos a criar interfaces limpas e responsivas (adaptáveis a celular), implementando menus do tipo 'hambúrguer' e navegação intuitiva para que qualquer pessoa, independente da afinidade com tecnologia, conseguisse marcar uma consulta.
